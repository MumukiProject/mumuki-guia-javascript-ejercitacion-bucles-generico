> Definí una función `repetir` que tome un valor `valor` y un número entero `cantidad`, y devuelva una array con `valor` repetido `cantidad` de veces. :repeat:
>
```javascript
ム repetir('lovelace', 3)
['lovelace', 'lovelace', 'lovelace']
ム repetir('a', 5)
['a', 'a', 'a', 'a', 'a']
ム repetir('html', 0)
[]
```
